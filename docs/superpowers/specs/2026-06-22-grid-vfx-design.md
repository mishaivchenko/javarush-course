# Grid & Visual Effects — Design Spec

## Goal
Виправити візуальні проблеми поточної сторінки: розширити сітку на всю ширину, ущільнити допоміжні панелі, додати ефекти першої/останньої коробки та кривавий екран поразки.

## Scope
Один файл: `lesson2_1/index.html`. Тільки CSS + renderer JS. Жодних змін логіки стратегій.

## Tasks

### 021 — Grid Expansion
**Проблема:** Сітка займає ~40% ширини, навколо чорна пустота.

**Рішення:** Прибрати `max-width: 560px` з `.grid`. Дозволити сітці рости до 100% ширини `.center-area` (за мінусом паддінгу). Збільшити `font-size` `.box` з 0.55em до ~1em щоб цифри читались на великих коробках.

### 022 — Compact Bottom & Path
**Проблема:** Path stripe (32px) + bottom bar (48px) = 80px на службову інформацію. Багато.

**Рішення:** Path stripe → 24px, bottom bar → 36px. Шрифти пропорційно зменшити. Мінімізувати паддінги.

### 023 — Fill Sidebar Void
**Проблема:** Під контролами в сайдбарі порожньо, особливо на високих екранах.

**Рішення:** Додати блок з інформацією про поточний раунд/стратегію або легенду (кольори = в'язні). Відцентрувати контент вертикально або заповнити нижню частину.

### 024 — First & Last Box Effects
**Проблема:** Відкриття першої коробки візуально нічим не відрізняється від будь-якої іншої.

**Рішення:**
- **Перша коробка:** яскрава біла/жовта спалах-анімація (0.3s), особливий звуковий візуал (box-shadow розширюється)
- **Остання коробка (found):** зелена корона або "✔" пульсація, яскравіше ніж звичайний found
- **Остання коробка (fail):** червоний спалах + тремтіння (shake-анімація) самої коробки

### 025 — Blood Lose Screen
**Проблема:** Поразка показується як звичайний текстовий оверлей "EXECUTED".

**Рішення:** Замінити стандартний LOSE-оверлей на:
- Червоний градієнтний оверлей (кривавий, від темно-бордового до яскраво-червоного)
- Анімовані "краплі" або "розводи" на весь екран (CSS псевдоелементи або canvas)
- Текст "EXECUTED" великими червоними літерами з ефектом крапель крові
- Легке тремтіння (shake) всього оверлею

## Success Criteria (загальні)
1. Сітка займає >80% ширини center-area
2. Path stripe + bottom bar займають <60px сумарно
3. Перша коробка спалахує при відкритті
4. Знайдена коробка має посилену анімацію
5. Програшна коробка тремтить
6. Екран поразки — кривавий, не просто текст
7. Жодних змін у логіці стратегій або симуляції
8. Жодних зовнішніх залежностей
