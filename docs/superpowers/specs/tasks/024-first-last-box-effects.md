# TASK_SPEC: 024 — First & Last Box Effects

## Ціль
Додати особливі візуальні анімації для першої та останньої коробки, які відкриває в'язень.

## Залежності
- 021-grid-expansion (фонова робота з grid)

## Критерії приймання
1. **Перша коробка:** яскравий спалах/розширення box-shadow (жовтий/білий), виділяється серед інших
2. **Остання коробка (found):** посилена зелена анімація, пульсація зі scale-ефектом
3. **Остання коробка (fail):** червоний спалах + shake/тремтіння коробки
4. Анімації не блокують виконання симуляції
5. Не впливають на продуктивність (<10 коробок за раз анімується)

## Реалізація
- CSS keyframes: `.box.first-box { animation: first-flash 0.4s }`, `.box.last-found { animation: crown-pulse 0.6s }`, `.box.last-fail { animation: fail-shake 0.5s }`
- JS: `renderBoxGrid()` — перевіряє `stepCount === 0` та `isLast && found === false`, додає класи
