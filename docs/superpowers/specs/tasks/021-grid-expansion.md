# TASK_SPEC: 021 — Grid Expansion

## Ціль
Прибрати обмеження ширини сітки. Сітка має займати всю доступну ширину між сайдбаром і правим краєм.

## Залежності
Немає.

## Критерії приймання
1. `.grid` не має `max-width` (або має 100%)
2. Box font-size: збільшений щоб цифри читались на великих коробках (0.55em → 0.9-1em)
3. Немає скролу на 1024×768+
4. Всі функції працюють
