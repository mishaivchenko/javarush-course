# TASK_SPEC: 021 — Grid Expansion

## Ціль
Збільшити сітку 10×10 пропорційно вікну, прибрати max-width обмеження, підняти font-size коробок, стиснути допоміжні панелі.

## Залежності
Немає (CSS only + мінімум JS).

## Критерії приймання
1. Сітка займає ~85% ширини center-area (було 560px фікс)
2. Box font-size: 0.85em (було 0.55em)
3. Box gap: 4px (було 3px)
4. Header висота: 40px (було 48px)
5. Path stripe висота: 28px (було 32px)
6. Bottom bar висота: 40px (було 48px)
7. Header h1 font-size: 1.3em (було 1.6em)
8. Sidebar: 165px (було 175px), mixer шрифти дрібніші
9. Немає скролу на 1024×768+
10. Всі функції працюють (стратегії, рендер, контроли)

## Реалізація
- CSS правки в :root секції (змінні)
- `.grid` max-width прибрати
- `.box` font-size збільшити
- По дрібниці підігнати padding/size елементів
