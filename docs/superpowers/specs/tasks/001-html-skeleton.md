# TASK_SPEC: 001 — HTML Skeleton

## Ціль
Створити базовий index.html у `lesson2_1/` з порожнім HTML5 каркасом, підключеним CSS та JS секціями.

## Залежності
Немає.

## Критерії приймання
1. Файл `lesson2_1/index.html` існує
2. Містить `<!DOCTYPE html>`, `<html>`, `<head>`, `<body>`
3. В <head> є `<style>` блок
4. В <body> є `<script>` блоки (3 шт: model, renderer, controller)
5. Відкривається в браузері без помилок консолі

## Обмеження
Один файл. Ніяких зовнішніх ресурсів.
