# TASK_SPEC: 013 — WIN / LOSE Dramatic States

## Ціль
При завершенні раунду показати драматичний візуальний стан WIN (всі в'язні вижили) або LOSE (хтось не знайшов).

## WIN
- Екран або банер з текстом "ALL SURVIVED"
- Зелене світіння (CSS box-shadow/text-shadow)
- Затримка ~2s перед наступним раундом
- Усі скриньки показують свої числа (state.revealed всі true)

## LOSE
- Екран або банер з текстом "EXECUTED"
- Червоний спалах/пульсація на всій панелі
- Шлях невдалого в'язня підсвічується темно-червоним
- Скриньки невдалого в'язня залишаються відкритими
- Затримка ~2s перед наступним раундом
- Решта скриньок залишаються закритими (не відкривати всі)

## Банер
- Позиціонується поверх сітки (absolute/relative)
- Напівпрозорий чорний фон
- Великий текст по центру
- Анімація появи (fade in або scale)

## Залежності
006 — Simulation loop, 002 — CSS theme

## Критерії приймання
1. WIN показує "ALL SURVIVED" із зеленим світінням
2. LOSE показує "EXECUTED" з червоним спалахом
3. Затримка ~2s перед наступним раундом
4. Банер зникає перед стартом нового раунду

## Обмеження
Чистий CSS + JS. Без gore/графічних зображень.
