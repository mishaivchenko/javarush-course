# TASK_SPEC: 025 — Blood Lose Screen

## Ціль
Замінити стандартний текстовий оверлей "EXECUTED" на кривавий, драматичний екран поразки.

## Залежності
Немає (CSS + renderOverlay.js).

## Критерії приймання
1. LOSE-оверлей має червоний градієнт (від темно-бордового до криваво-червоного)
2. На оверлеї присутні анімовані "краплі/розводи" (CSS-анімація або canvas)
3. Текст "EXECUTED" великий, червоний, з ефектом
4. Оверлей злегка тремтить (shake-анімація)
5. WIN-оверлей не змінюється

## Реалізація
- CSS: `.overlay.lose` — background з radial-gradient (криваві тони)
- CSS: `.blood-drip` псевдоелементи з анімованими краплями
- CSS: `.overlay.active.lose` з `animation: lose-shake`
- Можливе використання SVG-фільтрів для ефекту крові
