# TASK_SPEC: 010 — Run / Pause / Reset / Speed Controls

## Ціль
Кнопки керування симуляцією та слайдер швидкості.

## Кнопки
- **Run** (▶): `state.running = true`, викликає `runRound()` якщо не було запущено
- **Pause** (⏸): `state.running = false`
  - Симуляція зупиняється після поточного кроку (await sleep переривається)
- **Reset** (↺): `state.running = false`, скидає всю статистику, генерацію нової перестановки, render()
  - Якщо симуляція йде — зупиняється негайно

## Поведінка
- Run не запускає новий раунд якщо вже йде
- Pause не забороняє Run — можна відновити
- Reset скидає ВСЕ: статистику, історію, раунди
- Після Reset стан: `IDLE`, раунд 0, нова перестановка

## Speed Slider
- `<input type="range" min="10" max="300" value="50">`
- `state.speed = value` — затримка в ms між кроками
- Змінюється в реальному часі

## Залежності
006 — Simulation loop

## Критерії приймання
1. Run запускає симуляцію
2. Pause зупиняє
3. Reset скидає все
4. Speed впливає на швидкість негайно
5. Стан кнопок змінюється (Run/Pause неактивна коли відповідна дія неможлива)

## Обмеження
Чистий JS + HTML.
