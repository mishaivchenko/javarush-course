# TASK_SPEC: 002 — Dark Pixel-Prison CSS Theme

## Ціль
Додати в `<style>` повний CSS-темплейт: темна готична 8-bit в'язнична тема.

## Кольори
- `--bg`: `#0a0a0a`
- `--panel`: `#1a1a1a`
- `--stone`: `#2c2c2c`
- `--stone-light`: `#4a4a4a`
- `--text`: `#c8c8c8`
- `--text-dim`: `#666666`
- `--accent`: `#8b0000`
- `--accent-bright`: `#cc0000`
- `--found`: `#2a5a2a`
- `--found-text`: `#6aaa6a`
- `--fail`: `#3a1a1a`

## Стилі
- body: monospace шрифт, темний фон, білий текст
- Хедери: великі літери, ред-глоу тінь, піксельний вигляд
- Панелі: темні з `groove`/`double` бордерами, нерівні краї
- Сітка 10×10: CSS grid
- Скриньки (.box): темні, з бордером, мінімалістичні
- Кнопки: піксельні, з :hover/:active станами
- Слайдери: кастомні, темні, з червоним акцентом
- Заголовок "100 DEAD MEN": великий, розряджений, пульсуючий red glow (CSS animation)

## Залежності
001 — HTML skeleton

## Критерії приймання
1. Сторінка має темний фон #0a0a0a
2. Текст читається
3. Бордери виглядають піксельними
4. Red glow анімація на заголовку

## Обмеження
Тільки CSS. Жодних зображень.
