# TASK_SPEC: 008 — Prisoner Path Visualization

## Ціль
Візуалізувати шлях поточного в'язня: які скриньки він відкрив, які числа знайшов, чи продовжує ланцюжок.

## Поведінка
- Список `state.prisonerPath` містить індекси відкритих скриньок поточним в'язнем
- На сітці всі `.box.in-path` мають тьмяний червоний колір
- `.box.current` підсвічена червоним (остання відкрита)
- `.box.found` — якщо знайшов свій номер (зеленуватий)
- Після завершення в'язня (успіх або невдача) класи `.in-path` і `.current` очищаються
- Під сіткою текст: "Prisoner #12 — box #42 → found 7"

## Відображення ланцюжка
Під час відкриття:
```
Prisoner #12 — box #42 → 7 (continues)
Prisoner #12 — box #7 → 33 (continues)
Prisoner #12 — box #33 → 12 ✓ FOUND
```

## Залежності
007 — Box grid renderer, 006 — Simulation loop

## Критерії приймання
1. Поточна скринька підсвічена червоним
2. Шлях видно на сітці
3. Текст під сіткою оновлюється
4. При завершенні в'язня шлях очищається
5. При находженні — зелена підсвітка

## Обмеження
CSS + JS. Без анімацій руху.
