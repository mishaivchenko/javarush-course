# Lesson 1 — Project Rules

## Structure
- The entire project MUST be a single `index.html` file.
- No npm, no build tools, no backend.
- No local image files. Image assets are loaded from external URLs (Unsplash).

## Styling
- Movie posters use **background images from Unsplash** with dark overlays for readability.
- Use a **dark theme** with a **red accent color** throughout the UI.
- Unsplash attribution must be included in modals and footer.

## Coding Standards
- All CSS goes in a `<style>` block inside `index.html`.
- All JS goes in a `<script>` block inside `index.html`.
- Keep the HTML semantic and accessible where possible.
- No API keys or secrets in the source code.
